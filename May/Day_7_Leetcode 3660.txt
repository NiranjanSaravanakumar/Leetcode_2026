# 3660. Jump Game IX
3660. Jump Game IX

You are given an integer array nums.

From any index i, you can jump to another index j under the following rules:

Jump to index j where j > i is allowed only if nums[j] < nums[i].
Jump to index j where j < i is allowed only if nums[j] > nums[i].
For each index i, find the maximum value in nums that can be reached by following any sequence of valid jumps starting at i.

Return an array ans where ans[i] is the maximum value reachable starting from index i.

 

Example 1:

Input: nums = [2,1,3]

Output: [2,2,3]

Explanation:

For i = 0: No jump increases the value.
For i = 1: Jump to j = 0 as nums[j] = 2 is greater than nums[i].
For i = 2: Since nums[2] = 3 is the maximum value in nums, no jump increases the value.
Thus, ans = [2, 2, 3].

Example 2:

Input: nums = [2,3,1]

Output: [3,3,3]

Explanation:

For i = 0: Jump forward to j = 2 as nums[j] = 1 is less than nums[i] = 2, then from i = 2 jump to j = 1 as nums[j] = 3 is greater than nums[2].
For i = 1: Since nums[1] = 3 is the maximum value in nums, no jump increases the value.
For i = 2: Jump to j = 1 as nums[j] = 3 is greater than nums[2] = 1.
Thus, ans = [3, 3, 3].

 

Constraints:

1 <= nums.length <= 10^5
1 <= nums[i] <= 10^9

## Problem Explanation

You are given an integer array `nums`.

From any index `i`, you can jump:

* **Right side (`j > i`)** only if:

  [
  nums[j] < nums[i]
  ]

* **Left side (`j < i`)** only if:

  [
  nums[j] > nums[i]
  ]

For every index `i`, we need to find the **maximum value** reachable after making any number of valid jumps.

Return an array `ans` where:

```text
ans[i] = maximum reachable value starting from index i
```

---

# Intuition

Observe carefully:

* You can move right only to **smaller** values.
* You can move left only to **greater** values.

This creates groups of indices that are mutually reachable.

Inside one connected component:

* every index can eventually reach the **maximum element** of that component.

So the problem becomes:

> Divide the array into independent components and assign the maximum value of each component.

---

# Key Observation

We split components using:

* `prefixMax[i]` → maximum from `0 → i`
* `suffixMin[i]` → minimum from `i → n-1`

If:

\text{prefixMax}[i] \leq \text{suffixMin}[i+1]

then:

* all values on the left are smaller/equal than values on the right
* no valid jumps can connect both sides

So a component ends at index `i`.

---

# Approach

## Step 1 — Build Prefix Maximum

```text
prefixMax[i] = max(prefixMax[i-1], nums[i])
```

Example:

```text
nums = [2,3,1,5]

prefixMax = [2,3,3,5]
```

---

## Step 2 — Build Suffix Minimum

```text
suffixMin[i] = min(suffixMin[i+1], nums[i])
```

Example:

```text
suffixMin = [1,1,1,5]
```

---

## Step 3 — Find Component Boundaries

If:

```text
prefixMax[i] <= suffixMin[i+1]
```

then component ends at `i`.

For each component:

* compute its maximum
* assign that maximum to all indices inside it

---

# Dry Run

## Input

```text
nums = [2,3,1]
```

### prefixMax

```text
[2,3,3]
```

### suffixMin

```text
[1,1,1]
```

No partition occurs.

So the entire array is one component.

Maximum value:

```text

```

Answer:

```text
[3,3,3]
```

# Java Solution

```java
class Solution {
    public int[] maxValue(int[] nums) {

        int n = nums.length;

        int[] prefixMax = new int[n];
        int[] suffixMin = new int[n];

        // Build prefix maximum
        prefixMax[0] = nums[0];

        for (int i = 1; i < n; i++) {
            prefixMax[i] = Math.max(prefixMax[i - 1], nums[i]);
        }

        // Build suffix minimum
        suffixMin[n - 1] = nums[n - 1];

        for (int i = n - 2; i >= 0; i--) {
            suffixMin[i] = Math.min(suffixMin[i + 1], nums[i]);
        }

        int[] ans = new int[n];

        int start = 0;
        int currentMax = nums[0];

        for (int i = 0; i < n - 1; i++) {

            currentMax = Math.max(currentMax, nums[i]);

            // Component boundary
            if (prefixMax[i] <= suffixMin[i + 1]) {

                // Fill answers for this component
                for (int j = start; j <= i; j++) {
                    ans[j] = currentMax;
                }

                start = i + 1;

                if (start < n) {
                    currentMax = nums[start];
                }
            }
        }

        // Last component
        currentMax = nums[start];

        for (int i = start; i < n; i++) {
            currentMax = Math.max(currentMax, nums[i]);
        }

        for (int i = start; i < n; i++) {
            ans[i] = currentMax;
        }

        return ans;
    }
}
```

# Python3 Solution

```python
class Solution:
    def maxValue(self, nums: List[int]) -> List[int]:

        n = len(nums)

        prefixMax = [0] * n
        suffixMin = [0] * n

        # Build prefix maximum
        prefixMax[0] = nums[0]

        for i in range(1, n):
            prefixMax[i] = max(prefixMax[i - 1], nums[i])

        # Build suffix minimum
        suffixMin[n - 1] = nums[n - 1]

        for i in range(n - 2, -1, -1):
            suffixMin[i] = min(suffixMin[i + 1], nums[i])

        ans = [0] * n

        start = 0
        currentMax = nums[0]

        for i in range(n - 1):

            currentMax = max(currentMax, nums[i])

            # Component boundary
            if prefixMax[i] <= suffixMin[i + 1]:

                # Fill component answer
                for j in range(start, i + 1):
                    ans[j] = currentMax

                start = i + 1

                if start < n:
                    currentMax = nums[start]

        # Last component
        currentMax = max(nums[start:])

        for i in range(start, n):
            ans[i] = currentMax

        return ans
```
