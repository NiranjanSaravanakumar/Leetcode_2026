🧩 LeetCode 2839 - Check if Strings Can be Made Equal With Operations I
📘 Problem Summary


2840. Check if Strings Can be Made Equal With Operations II

You are given two strings s1 and s2, both of length n, consisting of lowercase English letters.

You can apply the following operation on any of the two strings any number of times:

Choose any two indices i and j such that i < j and the difference j - i is even, then swap the two characters at those indices in the string.
Return true if you can make the strings s1 and s2 equal, and false otherwise.

 

Example 1:

Input: s1 = "abcdba", s2 = "cabdab"
Output: true
Explanation: We can apply the following operations on s1:
- Choose the indices i = 0, j = 2. The resulting string is s1 = "cbadba".
- Choose the indices i = 2, j = 4. The resulting string is s1 = "cbbdaa".
- Choose the indices i = 1, j = 5. The resulting string is s1 = "cabdab" = s2.

Example 2:

Input: s1 = "abe", s2 = "bea"
Output: false
Explanation: It is not possible to make the two strings equal.
 

Constraints:

n == s1.length == s2.length
1 <= n <= 105
s1 and s2 consist only of lowercase English letters.
---

## 💡 Key Idea

* Indices `(0, 2)` form one group
* Indices `(1, 3)` form another group
* You can rearrange characters **within each group only**

---

# 🚀 Approach 1 (User Approach)

### 🔹 Idea

Check if each character in `s1` matches:

* Either the same index in `s2`
* Or its swappable pair `(i + 2) % 4`

### ⚠️ Limitation

This approach checks **position-wise matching**, but does NOT ensure correct **frequency matching within groups**, so it may fail in edge cases.

---

## ☕ Java Code

```java
class Solution {
    public boolean canBeEqual(String s1, String s2) {
        for(int i = 0; i < 4; i++){
            // check same position OR swapped position
            if(s1.charAt(i) != s2.charAt(i) &&
               s1.charAt(i) != s2.charAt((i + 2) % 4)) {
                return false;
            }
        }
        return true;
    }
}
```

---

## 🐍 Python Code

```python
class Solution:
    def canBeEqual(self, s1: str, s2: str) -> bool:
        for i in range(4):
            # check same position OR swapped position
            if s1[i] != s2[i] and s1[i] != s2[(i + 2) % 4]:
                return False
        return True
```

---

# 🚀 Approach 2 (Optimal Approach)

### 🔹 Idea

Instead of checking positions, compare **groups**:

* Even indices → `(0, 2)`
* Odd indices → `(1, 3)`

👉 Sort and compare both groups.

### ✅ Why it works

* Swaps don’t mix groups
* Only order changes inside each group
* So groups must have the same characters

---

## ☕ Java Code

```java
import java.util.Arrays;

class Solution {
    public boolean canBeEqual(String s1, String s2) {
        // even index group (0,2)
        char[] a1 = {s1.charAt(0), s1.charAt(2)};
        char[] a2 = {s2.charAt(0), s2.charAt(2)};
        
        // odd index group (1,3)
        char[] b1 = {s1.charAt(1), s1.charAt(3)};
        char[] b2 = {s2.charAt(1), s2.charAt(3)};
        
        Arrays.sort(a1);
        Arrays.sort(a2);
        Arrays.sort(b1);
        Arrays.sort(b2);
        
        return Arrays.equals(a1, a2) && Arrays.equals(b1, b2);
    }
}
```

---

## 🐍 Python Code

```python
class Solution:
    def canBeEqual(self, s1: str, s2: str) -> bool:
        return (
            sorted(s1[0::2]) == sorted(s2[0::2]) and
            sorted(s1[1::2]) == sorted(s2[1::2])
        )
```
