#Leetcode 3567

📌 Minimum Absolute Difference in Submatrices (LeetCode 3567)

🧠 Problem Statement

Given a 2D integer grid and an integer k, for every k x k submatrix, compute the minimum absolute difference between any two distinct elements inside that submatrix.

Return a 2D array result where:

result[i][j] represents the minimum absolute difference in the submatrix starting at (i, j).

If all elements in the submatrix are equal, return 0.



---

🚀 Approach

🔹 Key Idea

For every k x k submatrix:

1. Collect all elements.


2. Remove duplicates.


3. Sort the values.


4. Compute the minimum difference between adjacent elements.




---

⏱️ Time Complexity

O(m * n * k² log(k²))

Not optimal for large inputs (can be improved using advanced techniques like sliding window + balanced structures)



---

🧾 Java Solution

import java.util.*;

class Solution {
    public int[][] minAbsDiff(int[][] grid, int k) {
        int m = grid.length;
        int n = grid[0].length;

        int[][] result = new int[m - k + 1][n - k + 1];

        for (int i = 0; i <= m - k; i++) {
            for (int j = 0; j <= n - k; j++) {

                TreeSet<Integer> vals = new TreeSet<>();

                for (int r = i; r < i + k; r++) {
                    for (int c = j; c < j + k; c++) {
                        vals.add(grid[r][c]);
                    }
                }

                if (vals.size() == 1) {
                    continue;
                }

                int minAbsDiff = Integer.MAX_VALUE;

                Integer prev = null;
                for (int val : vals) {
                    if (prev != null) {
                        minAbsDiff = Math.min(minAbsDiff, val - prev);
                    }
                    prev = val;
                }

                result[i][j] = minAbsDiff;
            }
        }

        return result;
    }
}

🐍 Python 3 Solution

from typing import List

class Solution:
    def minAbsDiff(self, grid: List[List[int]], k: int) -> List[List[int]]:
        m, n = len(grid), len(grid[0])
        
        result = [[0] * (n - k + 1) for _ in range(m - k + 1)]
        
        for i in range(m - k + 1):
            for j in range(n - k + 1):
                
                vals = set()
                
                for r in range(i, i + k):
                    for c in range(j, j + k):
                        vals.add(grid[r][c])
                
                if len(vals) == 1:
                    continue
                
                sorted_vals = sorted(vals)
                
                min_abs_diff = float('inf')
                
                for x in range(1, len(sorted_vals)):
                    min_abs_diff = min(min_abs_diff, 
                                       sorted_vals[x] - sorted_vals[x - 1])
                
                result[i][j] = min_abs_diff
        
        return result
