#3296 Minimum Number of Seconds to Make Mountain Height Zero

You are given:

* An integer **`mountainHeight`** representing the height of a mountain.
* An array **`workerTimes`** where `workerTimes[i]` represents the time taken by the `i-th` worker to remove the **first layer** of the mountain.

### Worker Rule

If a worker takes `t` seconds for the first layer:

* 1st layer → `t`
* 2nd layer → `2t`
* 3rd layer → `3t`
* ...
* `k-th` layer → `k * t`

So the **total time** to remove `k` layers:

[
t \times (1 + 2 + 3 + ... + k)
]

Workers can work **simultaneously**.

### Goal

Find the **minimum number of seconds** required to reduce the mountain height to **0**.

---

# 💡 Key Insight

For a worker with time `t`, removing `k` layers costs:

[
t \times \frac{k(k+1)}{2}
]

We want the **minimum time** such that the **sum of layers removed by all workers ≥ mountainHeight**.

This suggests using **Binary Search on Time**.

---

# 🚀 Approach

### 1️⃣ Binary Search on Time

The answer lies between:

```
left = 1
right = max(workerTimes) * mountainHeight * (mountainHeight + 1) / 2
```

`right` is the **worst-case scenario** where the slowest worker removes all layers.

---

### 2️⃣ Check Function

For a given time `mid`:

For each worker:

* Simulate how many layers they can remove.
* Keep adding layers removed by each worker.
* Stop early if total ≥ mountainHeight.

---

### 3️⃣ Binary Search Decision

```
if check(mid):
    answer = mid
    search left half
else:
    search right half
```

---

# ⏱ Complexity

| Type              | Complexity              |
| ----------------- | ----------------------- |
| Binary Search     | O(log(maxTime × h²))    |
| Worker Simulation | O(n × √h)               |
| Total             | **O(n × √h × log(h²))** |

Where:

* `n` = number of workers
* `h` = mountainHeight

---

# 🧪 Example

### Input

```
mountainHeight = 4
workerTimes = [2,1]
```

### Explanation

Worker 1 (t=2):

```
1 layer → 2
2 layers → 2 + 4 = 6
3 layers → 2 + 4 + 6 = 12
```

Worker 2 (t=1):

```
1 layer → 1
2 layers → 3
3 layers → 6
```

If **time = 6 seconds**:

Worker 1 → 2 layers
Worker 2 → 3 layers

Total = **5 ≥ 4**

So the answer is:

```
6
```

# ☕ Java Solution

```java
class Solution {

    private boolean check(long mid, int[] workerTimes, int mountainHeight) {
        long total = 0;

        for (int t : workerTimes) {
            long time = 0;
            long layer = 1;

            while (time + layer * t <= mid) {
                time += layer * t;
                total++;
                layer++;

                if (total >= mountainHeight) {
                    return true;
                }
            }
        }

        return total >= mountainHeight;
    }

    public long minNumberOfSeconds(int mountainHeight, int[] workerTimes) {
        int maxTime = 0;

        for (int t : workerTimes) {
            maxTime = Math.max(maxTime, t);
        }

        long left = 1;
        long right = (long) maxTime * mountainHeight * (mountainHeight + 1) / 2;
        long ans = right;

        while (left <= right) {
            long mid = left + (right - left) / 2;

            if (check(mid, workerTimes, mountainHeight)) {
                ans = mid;
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }

        return ans;
    }
}
```

# 🐍 Python Solution

```python
class Solution:
    def check(self, mid: int, workerTimes: list[int], mountainHeight: int) -> bool:
        total = 0

        for t in workerTimes:
            time = 0
            layer = 1

            while time + layer * t <= mid:
                time += layer * t
                total += 1
                layer += 1

                if total >= mountainHeight:
                    return True

        return total >= mountainHeight


    def minNumberOfSeconds(self, mountainHeight: int, workerTimes: list[int]) -> int:
        maxTime = max(workerTimes)

        left = 1
        right = maxTime * mountainHeight * (mountainHeight + 1) // 2
        ans = right

        while left <= right:
            mid = (left + right) // 2

            if self.check(mid, workerTimes, mountainHeight):
                ans = mid
                right = mid - 1
            else:
                left = mid + 1

        return ans
```

# ⚡ Optimized Mathematical Approach

The previous approach simulates layers for every worker, which may take **O(√h)** per worker.

We can optimize this by **directly calculating how many layers a worker can remove in `mid` seconds** using a mathematical formula.

---

## 🔍 Derivation

For a worker with time `t`, removing `k` layers requires:

[
t \times (1 + 2 + 3 + ... + k)
]

Which simplifies to:

[
t \times \frac{k(k+1)}{2}
]

We need:

[
t \times \frac{k(k+1)}{2} \le mid
]

Rearranging:

[
k(k+1) \le \frac{2 \times mid}{t}
]

This forms a quadratic equation:

[
k^2 + k - \frac{2 \times mid}{t} \le 0
]

Solving using the quadratic formula:

[
k = \frac{-1 + \sqrt{1 + 8 \times mid / t}}{2}
]

Which simplifies in code to:

```
k = floor(sqrt(2*mid/t + 0.25) - 0.5)
```

Thus we can compute the number of layers **in O(1)** instead of simulation.

---

# 🚀 Optimized Approach

1. Use **Binary Search on Time**.
2. For each worker, compute the number of layers using the **quadratic formula**.
3. Sum all layers removed.
4. If total ≥ `mountainHeight`, the time is valid.

---

# ⏱ Complexity

| Method                | Complexity        |
| --------------------- | ----------------- |
| Simulation Approach   | O(n × √h × log h) |
| Mathematical Approach | **O(n × log h)**  |

Where:

* `n` = number of workers
* `h` = mountainHeight

The optimized approach is **much faster for large inputs**.

# ☕ Java Optimized Solution

```java
class Solution {
    private boolean check(long mid, int[] workerTimes, int mH) {
        long h = 0;

        for (int t : workerTimes) {
            h += (long)(Math.sqrt(2.0 * mid / t + 0.25) - 0.5);

            if (h >= mH) {
                return true;
            }
        }

        return h >= mH;
    }

    public long minNumberOfSeconds(int mountainHeight, int[] workerTimes) {
        int maxTime = 0;
        for (int t : workerTimes) 
            maxTime = Math.max(maxTime, t);

        long l = 1;
        long r = (long) maxTime * mountainHeight * (mountainHeight + 1) / 2;

        long result = 0;

        while (l <= r) {
            long mid = l + (r - l) / 2;

            if (check(mid, workerTimes, mountainHeight)) {
                result = mid;
                r = mid - 1;
            } else {
                l = mid + 1;
            }
        }

        return result;
    }
}
```
