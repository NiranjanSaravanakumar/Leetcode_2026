1594. Maximum Non Negative Product in a Matrix

You are given a m x n matrix grid. Initially, you are located at the top-left corner (0, 0), and in each step, you can only move right or down in the matrix.

Among all possible paths starting from the top-left corner (0, 0) and ending in the bottom-right corner (m - 1, n - 1), find the path with the maximum non-negative product. The product of a path is the product of all integers in the grid cells visited along the path.

Return the maximum non-negative product modulo 109 + 7. If the maximum product is negative, return -1.

Notice that the modulo is performed after getting the maximum product.

 

Example 1:


Input: grid = [[-1,-2,-3],[-2,-3,-3],[-3,-3,-2]]
Output: -1
Explanation: It is not possible to get non-negative product in the path from (0, 0) to (2, 2), so return -1.
Example 2:


Input: grid = [[1,-2,1],[1,-2,1],[3,-4,1]]
Output: 8
Explanation: Maximum non-negative product is shown (1 * 1 * -2 * -4 * 1 = 8).
Example 3:


Input: grid = [[1,3],[0,-4]]
Output: 0
Explanation: Maximum non-negative product is shown (1 * 0 * -4 = 0).
 

Constraints:

m == grid.length
n == grid[i].length
1 <= m, n <= 15
-4 <= grid[i][j] <= 4

You can only move:
- ➡️ Right
- ⬇️ Down

Return:
- The maximum non-negative product **modulo (10^9 + 7)**
- If no such product exists, return `-1`

---

## 💡 Approach (Dynamic Programming)

### 🔑 Key Insight
Because the grid contains **negative numbers**, we track:
- `max_dp[i][j]` → Maximum product up to cell `(i, j)`
- `min_dp[i][j]` → Minimum product up to cell `(i, j)`

> Negative × Negative = Positive → minimum can become maximum!

---

## ⚙️ Algorithm Steps
1. Initialize `max_dp` and `min_dp`
2. Fill first row and column
3. For each cell:
   - Compute 4 possibilities:
     - From top (max & min)
     - From left (max & min)
4. Update:
   - `max_dp[i][j] = max(...)`
   - `min_dp[i][j] = min(...)`
5. Return:
   - `-1` if result < 0
   - else `result % (10^9 + 7)`

---
````
#☕ Java Implementation
class Solution {
    public int maxProductPath(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;

        long[][] maxDp = new long[m][n];
        long[][] minDp = new long[m][n];

        maxDp[0][0] = minDp[0][0] = grid[0][0];

        // First column
        for (int i = 1; i < m; i++) {
            maxDp[i][0] = maxDp[i - 1][0] * grid[i][0];
            minDp[i][0] = maxDp[i][0];
        }

        // First row
        for (int j = 1; j < n; j++) {
            maxDp[0][j] = maxDp[0][j - 1] * grid[0][j];
            minDp[0][j] = maxDp[0][j];
        }

        // DP computation
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                long val = grid[i][j];

                long a = maxDp[i - 1][j] * val;
                long b = minDp[i - 1][j] * val;
                long c = maxDp[i][j - 1] * val;
                long d = minDp[i][j - 1] * val;

                maxDp[i][j] = Math.max(Math.max(a, b), Math.max(c, d));
                minDp[i][j] = Math.min(Math.min(a, b), Math.min(c, d));
            }
        }

        long result = maxDp[m - 1][n - 1];

        if (result < 0) return -1;

        return (int)(result % 1_000_000_007);
    }
}
````

## 🐍 Python Implementation

```python
class Solution:
    def maxProductPath(self, grid):
        m, n = len(grid), len(grid[0])
        
        max_dp = [[0] * n for _ in range(m)]
        min_dp = [[0] * n for _ in range(m)]
        
        max_dp[0][0] = min_dp[0][0] = grid[0][0]
        
        for i in range(1, m):
            max_dp[i][0] = max_dp[i - 1][0] * grid[i][0]
            min_dp[i][0] = max_dp[i][0]
        
        for j in range(1, n):
            max_dp[0][j] = max_dp[0][j - 1] * grid[0][j]
            min_dp[0][j] = max_dp[0][j]
        
        for i in range(1, m):
            for j in range(1, n):
                val = grid[i][j]
                
                a = max_dp[i - 1][j] * val
                b = min_dp[i - 1][j] * val
                c = max_dp[i][j - 1] * val
                d = min_dp[i][j - 1] * val
                
                max_dp[i][j] = max(a, b, c, d)
                min_dp[i][j] = min(a, b, c, d)
        
        result = max_dp[m - 1][n - 1]
        
        return result % (10**9 + 7) if result >= 0 else -1
````

Time Complexity: O(m × n)
Space Complexity: O(m × n)
